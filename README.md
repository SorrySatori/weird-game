# Weird Game

A 2D point-and-click adventure game built with Phaser.js.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:8080`

## Building for Production

1. Build the game:
```bash
npm run build
```

2. Serve the built files:
```bash
npm run serve-build
```

3. Open your browser and navigate to `http://localhost:8080`

The build process will create a `dist` directory with all the necessary files for deployment.

## Development

You can also use the webpack development server:
```bash
npm run serve
```

## Game Controls

- Use your mouse to interact with the game
- Click on objects to interact with them
